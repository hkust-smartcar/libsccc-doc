enum libbase::k60::FtmQuadDecoder::Config::EncodingMode
strong
Enumerator
kPhaseAB 

In this mode, the relationship between phase A and B signals indicates the counting direction, and phase A and B signals define the counting rate (FTM counter is updated when there is an edge either at the phase A or phase B signals)

kCountDirection 

In this mode, the phase B input value indicates the counting direction (FTM counter increment or decrement), and the phase A input defines the counting rate (FTM counter is updated when there is a rising edge at phase A input signal)
