uint8_t libsc::k60::UartDevice::Config::rx_irq_threshold = 1

The number of bytes in the Rx buffer needed to fire the interrupt.

This will affect how often new bytes are pushed to the internal buffer, or your listener being triggered, depending on the config. If a buffer is not available for your device, this setting has no effect

OnReceiveListener libsc::k60::UartDevice::Config::rx_isr

The listener for Rx events.

Return true if the listener has consumed the data. In that case, the data won't be pushed to the internal buffer

uint8_t libsc::k60::UartDevice::Config::tx_buf_size = 14

The size of the Tx buffer.

Old data will be poped when the buffer overflows. Notice that this size is not in bytes, but rather the number of Send* calls. Depending on the use case, the actualy buffer size in bytes will vary
