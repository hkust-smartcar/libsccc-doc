void libbase::k60::GpiArray::ConfigValueAsDmaSrc ( Dma::Config *  config)

Config this GpiArray up to be ready to serve as DMA source, and set config accordingly.

The following options are also set besides src:
Dma::Config::minor_bytes = 1~4 (depending on Config::count)
Dma::Config::major_count = 1

Notice that Gpi value is byte-aligned when working with DMA, so that if Config::start_pin is not the 0/8/16/24th pin, the data received would be pre-padded with data from other pins. Similarly, the data would be post-padded if Config::start_pin + Config::count does not fit in 1/2/3/4 byte boundary
