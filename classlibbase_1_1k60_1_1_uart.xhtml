void libbase::k60::Uart::ConfigTxAsDmaDst ( Dma::Config *  config)

Config this Uart up to be ready to serve Tx as DMA destination, and set config accordingly.

The following options are also set besides mux_src and dst:
Dma::Config::minor_bytes = 1

Tx IRQ will be disabled after invoking this method and needed to be enabled manually

Note
To use DMA, Config::tx_isr must NOT be set for this Uart. Otherwise, the operation will fail and no changes would be made to config
