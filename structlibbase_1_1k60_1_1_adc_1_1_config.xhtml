Adc::Name libbase::k60::Adc::Config::adc = Adc::Name::kDisable

Specify the pin to be used.

Either Config::pin or Config::adc has to be set. If Config::pin is set, Config::adc will be ignored. This provides two ways to specify the ADC pin. If the pin can be used with more than 1 ADC module (eg, PTB0, PTB1), the first one will be chosen by default. Otherwise you should specify the ADC pin through Config::adc
