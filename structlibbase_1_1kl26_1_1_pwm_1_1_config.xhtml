Precision libbase::kl26::Pwm::Config::precision = Precision::kUs

Set the unit of Config::period and Config::pos_width.

It is useful when the period is too small such that the precision is not high enough. Otherwise, the default should be kept

Note
Should only select Precision::kNs when period < 1000 us. Otherwise integer overflow may occur during some calculations
